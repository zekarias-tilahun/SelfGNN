# SelfGNN
A PyTorch implementation of SelfGNN: Self-supervised Graph Neural Networks without explicit negative sampling

Requirements!
-------------
  - Python 3.6+
  - PyTorch 1.6+
  - PyTorch Geometric 1.6+
  - Numpy 1.17.2+
  - Networkx 2.3+
  - SciPy 1.5.4+ 

Example usage
-------------


```sh
$ python src/train.py
```

or 

```sh
$ bash train.sh
```
